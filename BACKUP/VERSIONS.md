# История версий проекта Encoder-Only

## Version 0.1 (2025-10-10)

**Статус:** Стабильная рабочая версия

**Основные возможности:**
- Базовая работа с энкодером (вращение + кнопка)
- LED индикация (блинки по направлению вращения)
- Длинное нажатие с сбросом счетчика
- Параллельный вывод логов в UART1 и USB-CDC
- Архитектура на C++ (классы Led и Encoder)

**Файлы:** `Version_0.1/`

**Примечания:**
- Первая версия с поддержкой C++17
- Все исходники, CMakeLists.txt и конфигурация сохранены
- Проект готов к дальнейшему развитию

---

## Инструкция по восстановлению версии

Для возврата к версии 0.1:

```bash
# Из корня проекта
cp -r BACKUP/Version_0.1/Core ./
cp -r BACKUP/Version_0.1/USB_DEVICE ./
cp BACKUP/Version_0.1/CMakeLists.txt ./
cp -r BACKUP/Version_0.1/cmake ./
```

Затем пересобрать проект:

```bash
rm -rf build
mkdir build
cd build
cmake ..
make
```
